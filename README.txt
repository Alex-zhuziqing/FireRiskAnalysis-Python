This is a project exploring the bushfire risk with respect to land area, population, economic information and geometric information in Greater Sydney, NSW, Australia. 

Read dataset description, analysis method and result in 'report.pdf'.

The folder 'additional_dataset' contains the additional dataset. It has a subdirectory named 'NSW_BFPL_DATA' which is the original data downloaded from the NSW government website. The data after being cleaned is stored in 'RFSNSW_BFPL.geojson'. 

The script 'additional_data_cleaning_analysis.ipynb' is the script for cleaning and analyse the additional dataset.
